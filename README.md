# Análisis de Ventas Multitienda

## Descripción
Proyecto de análisis de datos para comparar el rendimiento de cuatro tiendas en términos de:
- Ventas por categoría
- Productos más vendidos
- Calificación promedio
- Ingresos totales
- Costos de envío
- Identificación de la tienda con menor desempeño para recomendar cierre

Se utilizaron datos históricos de compras, categorías de producto, ubicación, calificación de clientes y medios de pago.

## Objetivos
1. Calcular ventas por categoría en cada tienda.
2. Identificar productos más vendidos.
3. Determinar calificación promedio por tienda.
4. Medir ingresos totales por tienda.
5. Calcular costo de envío promedio considerando que el cliente lo paga.
6. Visualizar resultados con gráficos.
7. Analizar y recomendar el cierre de la tienda con menor desempeño.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
